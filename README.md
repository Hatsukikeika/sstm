# sstm
sstm每日自动签到

## 用法

```
usage: sstm签到工具
 -d,--ips4_device_key <arg>   cookie中的ips4_device_key
 -e,--message <arg>           你签到贴的内容
 -h,--help                    将此帮助消息输出到输出流
 -l,--ips4_login_key <arg>    cookie中的ips4_login_key
 -m,--ips4_member_id <arg>    cookie中的ips4_member_id
 -n,--hostname <arg>          如果要设置代理，代理的地址
 -p,--port <arg>              如果要设置代理，代理的端口
 -s,--ips4_ss_id <arg>        cookie中的ips4_ss_id
 ```
